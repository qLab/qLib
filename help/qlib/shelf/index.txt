:: [Icon:IMAGE/view_guides] [Toggle Guides|/shelf/qLib::ToggleViewportGuides] -- Toggle Guide geometry display

:: [Icon:BUTTONS/network_sticky] [Add Sticky Note|/shelf/qLib::AddStickyNote] -- Adds sticky notes with an user/timestamp header
:: [Icon:IMAGE/zoom_in] [Resize Sticky Notes|/shelf/qLib::ResizeStickyNotes] -- Enlarge/shrink selected sticky notes
